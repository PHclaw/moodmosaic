# MoodMosaic - Your Year in Color

> 每天选一个颜色，记录你的心情。一年后，拼出你的年度情绪色彩图。

## Features

- **Daily Mood Entry** - Pick from 8 moods with beautiful colors
- **Year Mosaic** - GitHub-style contribution graph in color
- **Weekly View** - This week at a glance
- **AI Insights** - DeepSeek-powered mood trend analysis
- **Statistics** - Visualize your mood distribution

## Tech Stack

- Backend: FastAPI + PostgreSQL + DeepSeek AI
- Frontend: React 18 + Vite + TypeScript + Tailwind CSS
- Deploy: Docker + Railway

## Quick Start

```bash
cd backend && pip install -r requirements.txt && uvicorn app.main:app --reload
cd frontend && npm install && npm run dev
```
